About Me

Hi — I'm the developer of this small ToDo project. I built this app to learn Flask, templating with Jinja2, and integrating a lightweight SQLite database with Flask-SQLAlchemy.

I enjoy building small, useful apps that help me practice full-stack basics: routing, templates, forms, and persistence. This project is intentionally simple and easy to extend.

MyToDo — Simple ToDo Manager

A minimal ToDo app implemented with Flask and Bootstrap. It demonstrates:

  • Flask routing and view functions
  • Jinja2 templates and template inheritance
  • Form handling and CRUD operations
  • Flask-SQLAlchemy with SQLite for storage

How it works

When you add a ToDo, the server saves it to a local SQLite database. The main page lists all saved todos and lets you update or delete items. The layout uses Bootstrap for responsive, clean design.

Ideas & Next Steps

Planned improvements include authentication, better form validation, due dates, and persisting user-specific lists. Contributions and feedback are welcome.
